# Book2Screen

Система для оцінювання та порівняння книг і їхніх кіноекранізацій.

## Технологічний стек
* **Frontend:** Vue 3 + TypeScript + Vite
* **Backend:** C# (.NET)

## Як запустити локально
1. Встановіть залежності: `npm install`
2. Створіть файл `.env` на основі `.env.example`
3. Запустіть сервер: `npm run dev`
